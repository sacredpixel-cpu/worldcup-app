# Local Development Setup

## Prerequisites
- Node 20+
- Expo CLI: `npm install -g expo-cli`
- iOS Simulator or Android Emulator (or Expo Go on device)

## Steps

```bash
git clone https://github.com/your-org/worldcup-2026.git
cd worldcup-2026
npm install
cp .env.example .env        # fill in your keys
npm run dev                 # starts Expo dev server
```

## Environment Variables
See `.env.example` for all required keys. Contact a team member for the sports API key.
