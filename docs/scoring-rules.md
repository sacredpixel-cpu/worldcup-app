# Scoring Rules

Points are awarded after each match result is confirmed.

| Prediction | Points |
|---|---|
| Correct outcome (win/draw) | **3 pts** |
| Exact scoreline | **5 pts** (includes outcome points) |
| Correct finalist | **4 pts** |
| Correct champion | **10 pts** |

Predictions lock at kick-off. Late submissions are not accepted.

## Bracket Predictions
Users predict the full bracket before the Round of 32. Points accumulate as rounds complete.
